# University Digital Transformation Project

## Overview
This project simulates the end-to-end delivery of a large-scale IT transformation within a Higher Education institution.

The project involves replacing a legacy Virtual Learning Environment (VLE) with a modern, cloud-based digital learning platform.

## My Role
Project Manager responsible for:

* Governance setup
* Stakeholder management
* Budget and resource planning
* Risk management
* Delivery oversight

## Objectives

* Improve student digital experience
* Enable scalable online learning
* Replace legacy systems

## Methodology

This project follows a structured project management approach aligned with PRINCE2 principles.

## Repository Structure

Each document in this repository represents a key project management deliverable.
